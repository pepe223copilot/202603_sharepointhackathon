04. 3 Features

FeaturesAI in SP / Agent① / Agent②
Powered by Three Core Features
AI in SharePoint

Knowledge Agent(AutoFill)+ Syntex
Invoice Data Extraction

  • Automatically extract data from invoices
  • Invoice Number / Issue Date / Due Date / Amount / Bill To
  • Consistency check with AutoFill / Syntex models
  • After validation → Trigger Agent ①
INPUT → PROCESS → OUTPUT
Invoice file → Extract & validate → Structured metadata
Copilot Studio

Agent ①
Match, Register & Validate

  • Match with customer master and register Customer ID
  • AI consistency check against billing recipient master
  • Human-in-the-loop final review
  • Update status to Human Checked
INPUT → PROCESS → OUTPUT
SharePoint + two external masters → Match & validate → Register Customer ID & update status
Copilot Studio

Agent ②
CSV Generation

  • Filter invoices due this month and marked Human Checked
  • Match with bank master to enrich account details
  • Group invoices by bank
  • Generate bank-specific CSV files
INPUT → PROCESS → OUTPUT
Approved invoices + bank master → Filter & group → Bank-specific CSV
These three features work together with SharePoint as the knowledge foundation, enabling a flow from "just drop it" to "payment ready"
