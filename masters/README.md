# Master Data

## Purpose

This directory organizes the schemas of external masters referenced by agents.

Invoice files alone do not provide sufficient information for business judgment — reference data for Customer ID, billing recipient details, and bank account information is required.

## Scope

- Customer Master
- Billing Recipient Master
- Bank Master

## Role

- Provides the matching basis for Agent ①
- Provides the CSV supplementation basis for Agent ②
- Supports business standards that cannot be held in SharePoint columns alone

## Notes

- Actual connection targets: environment-specific, not included
- Update operations: out of scope for this submission
- Data quality management rules: out of scope for this submission
